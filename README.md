## What this is for?

This command line tool helps you send newsletters.

## How to install?

Install it first:

```bash
$ gem install rumble
```

## How to run?

Run it locally and read its output:

```bash
$ rumble --help
```

## How to contribute?

Just submit a pull request. Make sure `rake` passes.
